MY PROJECTS:

"An PIR sensor based target detection and tracking using quadcopter"

The project was about making a smaller modular and commercially cheapertracking drone for security agencies to use for tracking and detection. Araspberry pi with passive infrared sensor and webcam is used for targetdetection and tracking. A cloud and viewer application is used to deploy livemonitoring feed on the mobile platform. The project was made to combat crossborder infiltration by the enemy..
